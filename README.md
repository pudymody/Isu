# Isu
A self hosted issuu pdf viewer generator from cmd. You can read more [here]()

# System requisites
* [GraphicsMagick](http://www.graphicsmagick.org/)
* [Ghostscript pdf interpreter](http://ghostscript.com/download/gsdnld.html)

# Installation
```sh
npm install -g isu
```

# Usage
```sh
isu --file=pdf.pdf
```
